**provider.ts-provider** is an open source project that includes developer toolkit for everyday tasks.

* Scale and convert formats for compression with point, bilinear or box filter.
* Convert from capture formats for processing.
* Convert to display formats for rendering.
* Rotate by 90/180/270 degrees to adjust for device orientation.
* Optimized for AVX2 on x86/x64.
* Optimized for SME on Arm.
* Optimized for MSA on Mips.
* Optimized for Vector on RISC-V.
